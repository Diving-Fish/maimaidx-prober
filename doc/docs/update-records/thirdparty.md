# 通过AWMC BOT进行成绩上传

使用第三方项目 AWMC BOT 通过官方数据源进行成绩上传。

您可以加入 QQ 官方群（1072033605）后，使用"mai绑定水鱼"命令进行更新。

> 初次使用需绑定账号，请根据 Bot 的教程使用相关命令进行绑定。仅需绑定一次后皆可使用相关功能。

# 通过Mizuki Bot进行成绩上传

使用第三方项目 **Mizuki Bot** 将您的成绩进行**水鱼+落雪**的双端上传。

目前Mizuki Bot 支持两种不同的模式

> **类型一**：通过友人~~(仇人)~~对战获取游玩成绩
>> **优点**：不登录账号，可保存/解绑用户UID，下次同步成绩无需发送二维码<br>
>> **缺点**：无法获取FC/FS状态，无法同步落雪收藏品状态

> **类型二**：登录用户账号获取游玩成绩
>> **优点**：相较于类型一，可获取FC/FS状态并同步落雪收藏品状态<br>
>> **缺点**：我们将登录账号并自动登出，在网络不佳等情况下，可能会导致账号**登出失败**，每次同步都需发送二维码，安全性有待商榷

> **开发者建议使用类型一进行同步，防止有心之人登录您的账号**
