# 开始使用

本查分器仅用于推分指导用途，不保证歌曲定数、DX Rating、难度 100% 准确。

## 使用指南

开始之前，您需要先注册一个查分器账户。

点击右上角的[查分器](https://www.diving-fish.com/maimaidx/prober/)链接，点击【登录并同步数据】->【立即注册】即可完成账户注册。

## 同意用户协议

> 
> 1. 注册和账号安全
>
> - 为了使用完整的账号功能，您需注册一个账号并同意用户协议。
> 
> - 您应对您的账号和密码负有保密责任，请勿将其提供给任何第三方。
> 
> - 如您发现有任何未经授权使用您账号的情况，应立即通知我们。
> 
> 2. 公开成绩信息风险
> 
> - 在本网站上公开您的成绩信息时，存在被不怀好意的攻击者利用并进行网络攻击的风险，成绩掩码并不能完全防止攻击者的攻击。

## 完成设置并生成导入 Token

登录状态下，点击页面右上角的【编辑个人资料】，会弹出如下面板。

除了必须 **点击下图中红框圈起来的图标** 生成一个 **成绩导入 Token** 用于成绩导入外，其他选项可以依靠您的喜好设置。

生成完毕之后，您可以点击 **该图标右侧的复制图标** 将它复制到剪切板中。

![](/img/intro-profile.png)

| 设置项 | 说明 |
| -- | -- |
| 昵称 | 你在查分器下使用的昵称，大部分可视化工具会使用它代替用户名显示 |
| 成绩导入 Token | 提供成绩的读写权限，**大部分情况下你应该使用它去查询或上传成绩** |
| 绑定 QQ 号 | 不安全，特供给 QQ 平台的机器人查询成绩用 |
| 绑定频道 ID | 不安全，特供给需要 QQ 开放平台唯一 ID 的相关工具查询成绩用 |
| 段位 | 装饰用 |
| 牌子设置 | 装饰用 |
| 禁止其他人查询我的成绩 | 勾选后，除登录网页查分器和 Token 以外的方案都无法获取成绩 |
| 对非网页查询的成绩使用掩码 | 勾选后，第三方工具获取的成绩数值将是一个不影响分数的最低值 |
